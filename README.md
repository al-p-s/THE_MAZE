# Что реализовано
----
## Сервер (этап 1 — почти полностью)

Генерация лабиринта (рекурсивный бэктрекинг, полная связность)
Размер поля по числу игроков
POI: Выход, Арсенал, Госпиталь, Клад
Спавн игроков по периметру
Туман войны (revealCell, revealWall, getPlayerView)
Система ходов: очередь, ОД, передача хода
Дебаффы W/S/P с тиком
Перемещение + столкновение со стеной
Проверка стены (actionCheckWall)
Проверка соседней клетки (actionCheckCell)
Атака по прямой (револьвер Пинкертона)
Урон, падение клада при ранении
Арсенал (+2 патрона или +2 бомбы)
Госпиталь (лечение до макс или аптечка)
Клад: выкопать, поднять, закопать, падение при уроне
Бомба: мина + взрыв внутренней стены
Аптечка
Выход с кладом → победа
Смерть игрока, дисконнект, победа последнего живого
Структура: state.js / actions.js / classes.js / maze.js


# Отложено
----

Базука (и дестрой клада)
Механики Пинкертона (радар 3x3, бесшумность, перебросить кубик)
Остальные 7 классов
Звуковые/текстовые подсказки по радиусу событий
PostgreSQL, история игр, статистика
Лобби, выбор персонажа
